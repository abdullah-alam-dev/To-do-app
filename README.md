# To-Do App

A simple, responsive, and interactive web application that allows users to manage their daily tasks efficiently. Users can add tasks, mark them as completed, and delete tasks. The app also saves tasks locally so that they persist even after refreshing the page.  

## Features
- Add tasks to your to-do list  
- Mark tasks as completed  
- Delete tasks easily  
- Tasks are saved in **localStorage** for persistence  
- Fully responsive design that works on mobile and desktop  

## Technologies Used
- **HTML5**: Semantic structure and content  
- **CSS3**: Styling, gradients, and responsive design with media queries  
- **JavaScript**: Dynamic task handling, event listeners, and local storage management  

## How to Use
1. Clone the repository or download the project files.  
2. Open `index.html` in any modern web browser.  
3. Type your task into the input box.  
4. Click **Add** to add the task to the list.  
5. Click on a task to mark it as completed (toggle check).
